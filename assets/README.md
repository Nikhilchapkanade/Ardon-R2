# Ardon-R2 — brand assets

Logo and visual identity files for the Ardon-R2 project.

## Files in this folder

| File | Purpose | Format / Size |
|---|---|---|
| `logo.png` | Master logo — flat "R2" mark with "Ardon" wordmark below | PNG, 1024×1024 or larger preferred |
| `logo.jpg` | Same as above, JPG fallback | JPG (avoid for logos if PNG available — JPG compression introduces artifacts on flat colors) |
| `social-preview.png` | GitHub repo social card (shows when the repo URL is shared on Twitter / LinkedIn / Slack) | PNG, 1280×640 — logo centered on white background |
| `favicon.png` | Browser tab icon | PNG, 64×64 or 32×32, just the R-mark (no wordmark) |

PNG is the recommended primary format. JPG is acceptable for the social-preview card but not for the logo itself (compression artifacts on flat solid colors).

A vector version (SVG) can be commissioned later for crisp rendering on Retina displays and arbitrary scaling. Not required for launch.

## GitHub repository settings to update after pushing

1. **Social preview image:** GitHub → repo → Settings → General → Social preview → upload `social-preview.png`
2. **Avatar:** if Ardon-R2 has its own GitHub organization, upload `logo.png` cropped square as the org avatar

## Usage rules

- Do not stretch, recolor, or rotate the mark.
- Maintain clear space around the logo equal to the height of the "R".
- For dark backgrounds, invert the image or use a recolored variant.
